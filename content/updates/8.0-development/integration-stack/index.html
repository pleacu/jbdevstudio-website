Red Hat JBoss Developer Studio 8.x - Integration Stack Updates

Build 8.0.7.GA

How Do I Install?


  1. Start JBDS. Go to Help > Install New Software... > Add...

  2. Paste in the URL of the update site at left. Browse the site for updates you'd like to install. Restart when prompted.

This site contains the latest Development Milestone Release Integration Stack updates for JBoss Developer Studio 8.x.

If you do not already have a copy of JBoss Developer Studio, click here to download it free.

Point your Red Hat JBoss Developer Studio at the following site:

  • Red Hat JBoss Developer Studio 8.x - Integration Stack :: For use with BPEL, BPMN2, jBPM, Drools, Fuse, JBoss ESB, Modeshape, SwitchYard, Teiid or JBoss WebServices
    • https://devstudio.jboss.com/updates/8.0-development/integration-stack/
    Also available for offline installation via zip:
    • jbdevstudio-integration-stack-updatesite-8.0.7.GA.zip
    • jboss-devstudio-8.1.0.GA-updatesite-core.zip
    • jboss-devstudio-8.1.0.GA-target-platform.zip (additional required dependencies)
